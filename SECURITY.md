# Security Policy

## Supported versions

Only the latest state of the `main` branch is supported. This is a fork intended
for building and rendering animations, not a published, versioned library.

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, use GitHub's private vulnerability reporting on this repository
(**Security → Report a vulnerability**). Describe the issue and, if possible,
steps to reproduce it. You'll get a response as soon as the maintainer is able.

For vulnerabilities in Motion Canvas itself, please report them upstream at
<https://github.com/motion-canvas/motion-canvas>.
